# 동적 할당이란?

***

우리가 지금까지 배우고 사용한 것, 예를 들면 배열 선언은 정적 할당이다.

동적 할당은 배열에서 많이 사용한다. 동적 할당은 메모리의 할당을 동적으로 해주는 것이다.

데이터 영역 스택 영역(데이터스택): 메모리를 할당할 영역이 컴파일 단계<sup>[1](#컴파일-단계)</sup>에서 작동
- 대표적인 것: 배열의 인덱스
변수라던지 그런 것들 -> 런타임 영역<sup>[2](#런타임-영역)</sup>에서 할당됨
